# Claude Rules for this repository

## Language

- Write all documentation (README, etc.) in English
- Write all PR titles and descriptions in English
- Write all commit messages in English

## Development Tools

See `mise.toml` for available tools and tasks.

### Common Tasks

- `mise run fmt` - Format all files
- `mise run fmt:check` - Check formatting
- `mise run build` - Build all CLIs
- `mise run test` - Run unit tests
